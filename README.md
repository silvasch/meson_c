# Meson C

A template for c projects configured using meson and just.

## Dependencies

* a c compiler
* meson
* just

## Usage

1. Setup or reconfigure the project:

```bash
just setup
```

2. Compile the project:

```bash
just build
```

3. Run the project:

```bash
just run
```

You can also pass arguments to the compiled binary:

```bash
just run "Planet Eearth"
```
